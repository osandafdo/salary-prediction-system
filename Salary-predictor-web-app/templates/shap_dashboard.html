Understanding model predictions using SHAP (SHapley Additive exPlanations)

About SHAP Values

SHAP values show how each feature contributes to the model's prediction for individual instances. Positive SHAP values increase the prediction, while negative values decrease it.

Feature Importance Summary

This plot shows which features are most important for the model's predictions.

Mean Absolute SHAP Values

This bar plot shows the average impact of each feature on the model output.

Waterfall Plot for First Instance

This plot shows how each feature contributes to the prediction for a specific instance.

SHAP values provide a unified measure of feature importance that is consistent with human intuition.
